## Description
[NestJS](https://github.com/nestjs/nest) framework (GraphQL/TypeScript) example of APIs that will enable a client to perform CRUD operations on the contact collection.

## Installation
```bash
$ npm install
```

## Running the app -- localhost:3000/contacts
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test
```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
